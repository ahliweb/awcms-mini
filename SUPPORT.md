# Dukungan / Support

Terima kasih sudah memakai AWCMS-Mini. Halaman ini menjelaskan cara mendapatkan bantuan.

## Sebelum bertanya

1. Baca [`README.md`](README.md) untuk gambaran umum dan status repositori.
2. Baca [`AGENTS.md`](AGENTS.md) untuk aturan kerja dan arsitektur.
3. Telusuri paket dokumen di [`docs/awcms-mini/`](docs/awcms-mini/README.md) — kemungkinan besar jawabannya sudah terdokumentasi.
4. Cari di [issue yang sudah ada](https://github.com/ahliweb/awcms-mini/issues) (open & closed).

## Kanal bantuan

| Kebutuhan                        | Kanal                                                                                                               |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Pertanyaan umum / diskusi        | [GitHub Discussions](https://github.com/ahliweb/awcms-mini/discussions) (bila aktif) atau issue berlabel `question` |
| Laporan bug                      | Buka issue dengan template **Bug report**                                                                           |
| Usulan fitur                     | Buka issue dengan template **Feature request**                                                                      |
| Perbaikan/permintaan dokumentasi | Buka issue dengan template **Documentation**                                                                        |
| **Kerentanan keamanan**          | **Jangan** buka issue publik — ikuti [`SECURITY.md`](SECURITY.md)                                                   |

## Ekspektasi respons

Ini adalah proyek yang dikelola secara best-effort. Waktu respons dapat bervariasi. Untuk mempercepat:

- Sertakan versi, langkah reproduksi, dan output yang relevan (tanpa secret/data asli).
- Satu issue untuk satu masalah.
- Gunakan judul yang deskriptif.

## Berkontribusi balik

Perbaikan lewat Pull Request sangat dihargai — lihat [`CONTRIBUTING.md`](CONTRIBUTING.md).
